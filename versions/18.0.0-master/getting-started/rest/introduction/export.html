Part 13: Exports

Export items from a case.

Legal Export

Let’s go ahead and do a legal export of our production set. We will use the production set ID

59d69780-963a-478c-9f4b-0d2bb4c84a3f

returned from the previous create production set task.

For this example, we will create a Ringtail (now, Nuix Discover®), load file and export both natives and PDF files.

Item Export

We can also export items using a query. Let’s export the HTML files from this case. For this example, we will export both natives and PDF files.

Request Response 
curl --location --request PUT 'http://localhost:8080/nuix-restful-service/svc/v1/cases/43b070164ce8453ca30ed9e2dfcce67b/export' \
--header 'nuix-auth-token: 9729a460-eda7-48dc-ba70-d12b3aae3c8d' \
--header 'Content-Type: application/json' \
--header 'Accept: application/json' \
--data-raw '{
  "exportType": "ITEM",
  "path": "/Exports/itemExport",
   "productTypes": [
    "NATIVE",
    "PDF"
  ],
  "queries": [
    "tag:HTMLFiles"
  ],
  "parallelProcessingSettings": {
  	"workerCount": 1
